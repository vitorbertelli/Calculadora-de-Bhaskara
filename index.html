Como funciona a equação do segundo grau

A equação do segundo grau, ou também conhecido como função quadrática, possui o termo de maior grau elevado ao quadrado, por essa razão é chamada assim. A equação é representada por:

ax² + bx + c = 0

Nessa equação, os valores representados pelas letras A, B e C são os coeficientes da equação, e o X é a incógnita.

Para ser considerado uma equação do segundo grau, o coeficiente A, acompanhado pela incógnita elevada ao quadrado, tem que ser diferente de zero.

Passo a Passo

Para resolver a equação usando a fórmula de Bhaskara, devemos seguir os seguintes passos:

  • 1º Passo: Indentificar os coeficientes.

Para facilitar o reconhecimento dos coeficientes, que podem aparecer em ordens diferentes, conclui-se que o coeficiente A sempre estará junto ao x², o B seguido do x e o C é o termo sem o x.

  • 2º Passo: Calcular o delta.

Para calcularmos a fórmula de Bhaskara, precisamos definir o valor do delta da equação do segundo grau. Por esse motivo utilizamos a próxima fórmula:

Δ = b² - 4.a.c

Agora, a partir do valor do delta, calculamos a raiz quadrada do mesmo. Através do valor do delta, podemos concluir as possíveis situações abaixo:

  • Δ < 0 = A equação não possui raízes reais.
  • Δ = 0 = A equação possui apenas uma raiz real.
  • Δ > 0 = A equação possui duas raízes reais.
  • 3º Passo: Calcular as raízes.

Com o valor de delta maior ou igual a zero, para descobrirmos os valores das raízes, basta calcular a seguinte fórmula de Bhaskara:

X = (-b ± √Δ) / 2.a
